# Shop Details Feature

This feature handles the display of shop profiles and their offerings.

## Structure

- `screens/`: Contains `ShopDetailsScreen.tsx`.
- `components/`: Contains shop-specific components.

## Usage

Accessed via `/shop/[id]`.
Displays shop information, products, services, and contact options.

## Dependencies

- `expo-router`
- Shared UI components (`StaticHeader`, `NeoButton`)
- `services` feature integration (for booking)
